# Quick File Picker

# How to use
Hit `CMD + Shirft + P` and input 'wwm', you'll see:
```
> filepicker: Search Files (Cmd+P on mac or Ctrl+P on linux/windows)
> filepicker: Build File List Database
```

Build file list database, then search for files, that's it!
